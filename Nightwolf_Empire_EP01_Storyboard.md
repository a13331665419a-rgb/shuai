# 夜狼帝国 Nightwolf Empire — EP01 导演逐镜头分镜脚本

## 剧本解读与节拍拆解

**本集核心戏剧目的**：建立世界观权力结构 + Nicholas的绝对王权 + Violet与Nicholas命运般的初遇 + Kylan当众提出离婚的羞辱炸弹

**题材类型**：女频强情绪 + 狼人权力 + 危险吸引 + 身份压制

**核心驱动力**：权力压制 → 危险吸引 → 公开羞辱

**节拍拆解**：
1. 帝国权力建立（仪式感、压迫感）
2. Nicholas王权亮相（危险、克制、绝对控制）
3. 四Alpha效忠（群像压迫、权力阶梯）
4. Violet入场（脆弱进入权力空间、反差）
5. 命运相遇——目光锁定（VFX红眼、狼影、Fated Bond初触）
6. Kylan当众提出离婚（全场震惊、羞辱炸弹、卡点）

**镜头重心**：冲突 + 危险吸引 + 身份压制

**预估镜头数**：80–100镜（高冲突 + 群像 + VFX + 权力建立 + 强卡点）
