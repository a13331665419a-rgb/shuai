## Part B: 台词分析卡

---

**L-01**
```
台词编号    ：L-01
说话者      ：SAM
台词原文    ：陛下，山地狼群的 Alpha 请求觐见。
字面含义    ：转达觐见请求
潜台词      ：我在探测你的边界；我知道你的回答很可能是"不"
情绪状态    ：职业克制，内心有一层谨慎
对听者意图  ：让 Nicholas 觉得这是例行汇报，而非挑战
关键词/停顿 ：「陛下」（称谓有停顿，礼节性节奏）；「请求觐见」（结尾降调，期待回应）
平行任务    ：Sam 边走向辅臣位边说，脊背始终极直，不用视线压迫 Nicholas
听者预期反应：Nicholas 面部无变化，但右手食指轻抬一次
景别策略    ：SAM 中景（平视，职业感）；NICHOLAS 反应用大特写（权力不对等）
```

**L-02**
```
台词编号    ：L-02
说话者      ：NICHOLAS
台词原文    ：Mountain Pack，没有资格出席这场议会。
字面含义    ：拒绝
潜台词      ：我不用解释，权力本身就是理由
情绪状态    ：绝对的、不需证明的冷漠
对听者意图  ：向殿下所有人（不止 Sam）宣示：我的决定不可撼动
关键词/停顿 ：「没有资格」（三字，落地清脆，斩钉截铁）
平行任务    ：Nicholas 说这句话时视线不看 Sam，看向殿下某个方向——他在对整个空间说话，不是对某个人
听者预期反应：殿下臣子低头；Sam 面部无变化但眼皮轻眨一次（他还没放弃）
景别策略    ：NICHOLAS 低仰拍中近景（权力感）；臣子们俯拍过肩（渺小）
```

**L-03**
```
台词编号    ：L-03
说话者      ：SAM
台词原文    ：他的 Luna 也到了。她的父亲是 Ocean Pack 的 Alpha。三年前，那场婚约是由您亲自认可的。
字面含义    ：补充信息，引入 Violet 的身份
潜台词      ：我在用你自己认可的东西来推翻你的决定——这不是我的权力，这是你给自己的限制
情绪状态    ：表面克制的进言，内心是精准的进逼
对听者意图  ：在权力的盔甲上找到一道缝隙并楔入——"三年前您亲自认可的"是楔子
关键词/停顿 ：「她的父亲」（主动拉入新角色）；
              「三年前」（时间锚点，关键词，必须插入听者反应）；
              「是由您亲自认可的」（强调主语是 Nicholas，不是 Sam）
平行任务    ：Sam 在第一个逗号后微微上前半步，动作比台词更先表达"进言"的决心；
              说"三年前"时，视线从 Nicholas 前方地面移至 Nicholas 脸部——唯一一次试探性直视
听者预期反应：Nicholas 的眼皮在"三年前"这个词上停顿了零点几秒（唯一暴露内心的身体反应）
景别策略    ：SAM 双人构图前景浅焦（进言的主动性）；NICHOLAS 在台词节点"三年前"切入大特写
```

**L-04**
```
台词编号    ：L-04
说话者      ：NICHOLAS
台词原文    ：那个生来没有狼的 Alpha 之女。
字面含义    ：对 Violet 身份的陈述/确认
潜台词      ：我记得她——不是因为婚约，而是因为她本身。"生来没有狼"是一个令我注意的特质。
情绪状态    ：表面是冷漠的信息确认，内层有一丝极淡的、旁人无法解读的回忆色彩
对听者意图  ：表面是对 Sam 的回应，实际是 Nicholas 在与自己的记忆对话
关键词/停顿 ：「生来没有狼」（这四字是全句的核心，Nicholas 在这四字上有极微妙的停顿）
平行任务    ：Nicholas 说这句话时视线投向远处虚空，不看任何人——这是他在回忆的姿势
听者预期反应：Sam 眨眼（他在猜测 Nicholas 此刻的心思）；殿下无反应
景别策略    ：NICHOLAS 中近景低仰拍，视线方向是虚空（这与 L-02 视线方向不同，暗示情绪变化）；
              Sam 的反应用平视中近景（不再是低仰，权力差异在这一刻微妙缩小了一点）
```

**L-05**
```
台词编号    ：L-05
说话者      ：NICHOLAS
台词原文    ：让他们进来。
字面含义    ：指令，允许觐见
潜台词      ：我选择了——不是因为规则，而是我想见她（但我绝不会承认这一点）
情绪状态    ：决断回归，掌控感重新建立，但这个决断背后有一层不被承认的动因
对听者意图  ：对 Sam——我的决定是我的决定，与你的进言无关
关键词/停顿 ：「让他们进来」（三字，语气平静，没有任何表情，但这是全场的关键转折点）
平行任务    ：Nicholas 在说这三个字时，视线终于回到正前方。眼皮轻轻降下，是王的表态动作
听者预期反应：Sam 微微敛眉然后平复（他达到了目的，但不表现出来）；J-Cut——"进来"声音先于门开
景别策略    ：NICHOLAS 大特写（最强权力感）；这一镜就像一把落锤
```
