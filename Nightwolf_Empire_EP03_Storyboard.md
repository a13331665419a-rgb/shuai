# 夜狼帝国 Nightwolf Empire — EP03 导演逐镜头分镜脚本

## 剧本解读与节拍拆解

**本集核心戏剧目的**：通过婚礼闪回揭示Kylan的背叛 → 建立Violet的核心创伤 → 回到走廊Violet彻底反击宣战

**题材类型**：女频强情绪 + 闪回羞辱 + 公开背叛 + 觉醒反击

**核心驱动力**：情感背叛 → 公开羞辱（过去时） → 爽感反击（现在时）

**节拍拆解**：
1. 闪回开始——三年前婚礼现场（温暖→破碎）
2. 新娘Violet满怀期待走红毯
3. 等在红毯尽头的不是Kylan而是Beta Maro——灵魂一击
4. Rex暴怒、宾客嘲讽、戒指被踩碎——多重羞辱叠加
5. 闪回结束——Violet的内心独白总结创伤
6. 回到走廊——Kylan的狡辩 vs Violet的碾压式反击
7. Violet离场——独白对月亮女神祈祷

**预估镜头数**：65–80镜
