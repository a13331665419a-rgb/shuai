# Skill: Dialogue Shot Sequence Designer（对话分镜序列设计师）

## Description
本技能为对话场景的专项精修工具。在 `director-storyboard-core.md` + `director-storyboard-advanced.md` 的基础上，对含台词的所有场次进行**逐句、逐词、逐停顿**的镜头序列设计。输出不止是拆镜，而是将对话的权力结构、潜台词、情绪呼吸、肢体语言、环境反应融为一体的**可执行视觉乐谱**。

## Prerequisites
- 必须先完成第一层分镜脚本（director-storyboard-core.md 输出）
- 推荐叠加 director-storyboard-advanced.md Module D
- 输入为：剧本原文对话 + 第一层分镜表中对应场次的镜头
- 本技能产出**覆盖并精化**第一层分镜表中的对话镜头，不是追加，而是替换

## Core Concepts（核心概念）

### 对话的三个层次
```
表层（Surface）  ：台词字面意思
中层（Subtext）  ：角色真正想表达的/不想表达的
深层（Dynamics）：对话中权力的流动方向
```
三层同时设计——表层给声音，中层给眼神/肢体，深层给构图/景别/运镜。

### 对话镜头链标准（Dialog Shot Chain）
每句台词默认分解为以下链条，可根据情绪密度增减节点：

```
①  起势镜头（Setup）          ——说话者开口前的状态
②  说话者A首节台词             ——说话者特写/中近景
③  听者B的即时反应             ——听者微表情特写
④  说话者A续句（含停顿/逗号）  ——景别/角度必须与②不同
⑤  环境/第三方反应             ——"沉默的空气"镜头
⑥  说话者A台词落地             ——高潮句，最强视觉冲击
⑦  半秒留白（After Beat）     ——台词结束后的情绪余震
```

单句短台词 → 最少使用 ①②③⑦（四镜头链）
长句/宣言台词 → 可扩展至七节点完整链
内心独白（V.O.）→ 不需说话者开口动作，画面应承载独白无法承载的信息

### 权力地图（Power Map）
对每一段对话，首先绘制权力流向：

```
角色A ──[权力方向]──► 角色B
```
- 权力高者：景别偏低仰，构图偏宽，空间占据多
- 权力低者：景别偏高俯或OTS后景，构图偏窄，空间被压缩
- 权力转移瞬间 → 景别/角度突变是最有效的视觉化手段

### 潜台词可视化（Subtext Visualization）
```
潜台词类型        视觉策略
─────────────────────────────────────
「我不相信你」   → 听者视线方向轻微偏移（不正面）
「我知道你在说谎」→ 听者视线在说话者嘴部停留（遮挡镜头）
「我想说但不能说」→ 说话者视线看向第三方/中空/下方
「我正在隐藏情绪」→ 吞咽 / 握拳 / 嘴唇抿紧 / 瞳孔收缩
「我已经知道结果了」→ 无微表情变化的静止特写（最恐怖的听者反应）
「这句话伤到我了」→ 2帧的震惊凝固，然后强行恢复表情
```

### 沉默镜头（Silence Shot）
对话中每一个沉默/停顿/欲言又止，至少设计一个专属镜头：
- 长沉默（>2秒）→ 一个环境细节镜头 + 一个手部/姿态特写
- 短停顿（逗号级）→ 听者的微表情切入（0.5-1秒）
- 欲言又止 → 说话者嘴唇微动又闭合的特写，**不加音效**

### 平行任务设计（Parallel Action Design）
角色说台词时必须有同步进行的"平行任务"：

```
台词情绪           平行任务设计原则
─────────────────────────────────────
威权/命令          → 说话者同时进行与对话无关的动作（翻看文件/看远方），强化漠然
恳求/进言          → 说话者动作收缩（靠近、上前半步），身体语言在说话
愤怒（压制中）     → 说话者用其他动作"代替"愤怒（整理领口/攥紧手）
惊慌但表面克制     → 说话者的手在身侧/背后细微颤抖
情绪失控           → 动作大于台词——台词是次要的，肢体是主要的
```

## Execution Flow（执行流程）

### Phase 0: 对话段落识别
- 从分镜脚本中标记所有含台词的场次
- 为每段对话命名（如 "EXCHANGE-01：觐见请求"）
- 统计台词数量、说话角色、情绪弧度

### Phase 1: 权力地图绘制
- 对每段对话绘制权力地图（见 Core Concepts）
- 标注权力转移节点（台词编号）
- 预判景别策略

### Phase 2: 逐句台词分析
对每句台词填写分析卡：

```
台词编号    ：L-01
说话者      ：
台词原文    ：
字面含义    ：
潜台词      ：
情绪状态    ：（说话者内心真实状态）
对听者意图  ：（试图让听者产生什么感受）
关键词/停顿 ：（标注逗号、省略号、情绪高潮词）
平行任务    ：（说话者此刻在做的其他动作）
听者预期反应：（听者应有什么外在表现）
景别策略    ：（基于权力地图）
```

### Phase 3: 逐句镜头链设计
- 依照 Dialog Shot Chain 七节点为每句台词设计完整镜头序列
- 输出标准8列分镜表，镜头编号格式为 D[EXCHANGE编号]-[序号]（如 D1-001）
- 必须包含：起势 + 台词主体 + 听者反应 + 环境反应 + 留白

### Phase 4: 沉默镜头补全
- 将所有对话间的沉默/停顿单独列出
- 为每个沉默点设计1-2个专属镜头

### Phase 5: 对话段节奏图
- 以时间轴形式标注整段对话的情绪弧度
- 标出高潮点、低谷点、节奏加速/减速段

### Phase 6: 自检
1. 每句台词是否均完成了七节点链中至少四点？
2. 听者反应镜头是否均先于/重于说话者镜头？
3. 权力地图的视觉化是否落地到景别/构图设计中？
4. 是否有遗漏的潜台词可视化机会？
5. 沉默点是否均有专属镜头设计？
6. 平行任务是否已为每个说话者设计？
7. 正反打是否在角度/景别/焦距上全程持续变化？

## Output Format（输出格式）

### Part A: 对话分析总表
```
EXCHANGE-[编号]：[对话名称]
═══════════════════════════════════════
涉及角色     ：
台词总数     ：
权力地图     ：角色A ──[方向]──► 角色B（可多方向、可双向）
权力转移节点 ：L-XX 台词落地时
情绪弧度     ：[起始情绪] → [高潮情绪] → [结束情绪]
核心潜台词   ：
对话目的     ：
═══════════════════════════════════════
```

### Part B: 台词分析卡（每句一卡，不可省略）

### Part C: 逐句镜头链分镜表（8列标准格式）

| 镜头编号 | 关联台词 | 景别/视角/构图 | 画面描述 | 情绪&节奏 | 声音设计 | 衔接逻辑 | 运镜提示 |
|---------|---------|--------------|---------|----------|---------|---------|---------|

### Part D: 沉默镜头表（独立列出）

### Part E: 对话节奏图（文字版时间轴）

## Output Rules
- 台词分析卡严禁省略任何一句
- 镜头链中"听者反应"镜头的画面描述字数不得少于"说话者"镜头
- 所有画面描述中的情绪词必须转化为物理描述（遵循可视化铁律）
- 沉默点镜头独立成段，不折叠进台词链
- 每段对话输出完毕后立即执行 Phase 6 自检，附在段落末尾

---

## Guiding Principles
- 台词是剧情，沉默是情感——听者反应比说话者台词重要十倍
- 权力是空间——谁占据了更多画面空间，谁拥有权力
- 潜台词是电影的灵魂——它只存在于画面，不存在于台词
- 正反打不是乒乓球——每一次切换都必须让观众的情绪发生微小的移动
- 对话的节奏是呼吸——有吸有呼，有涌入有释放
- 最好的对话镜头，是即使你不懂这门语言也能看懂角色关系的镜头
