# 夜狼帝国 Nightwolf Empire — EP05 导演逐镜头分镜脚本

## 剧本解读与节拍拆解

**本集核心戏剧目的**：EP04下半段（喝血规则建立）→ Kylan带Lucy到月亮祭坛 → Violet和Nicholas被迫藏在桌下 → 暧昧窒息的身体接触 → Violet手放在Nicholas裆部 → Kylan差点掀开桌布——EP05强卡点

**题材类型**：女频强情绪 + 暧昧拉扯 + 荒诞喜感 + 情色擦边

**核心驱动力**：危险吸引 → 暧昧身体接触 → 荒诞喜剧 → 悬念危机
