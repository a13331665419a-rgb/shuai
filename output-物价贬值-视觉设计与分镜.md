# Part I: 视觉设计规范

---

## A. 全局视觉风格指南

```
【作品视觉DNA】
- 视觉风格：半写实动漫风（韩漫质感），Q版变形用于喜剧插入
- 主色调：冷灰蓝 #4A5568（前半段/压抑）→ 暖琥珀金 #D69E2E（后半段/逆转）
  全片整体色彩弧线：低饱和冷调 → 随系统激活逐步暖化 → 爽感场景饱和度拉满
- 辅助色：都市银灰 #A0AEC0 / 混凝土灰白 #E2E8F0 / 深木棕 #5D4037
- 点缀色：
  · 系统蓝 #00B0FF — 仅限系统UI弹出时使用
  · 财富金 #FFD700 — 仅限钞票发光、金额显示、关键道具高光
  · 危险红 #FF1744 — 仅限角色情绪爆发/冲突升级标记
  使用规则：三种点缀色在非触发状态下严禁出现，保证触发时的视觉冲击力
- 光影风格：前半段高对比戏剧光（压抑段落）→ 逆转后转为高调明亮光（爽感段落）
- 材质倾向：现代都市混合（混凝土+玻璃幕墙+不锈钢 / 旧物粗糙质感对比高档材质）
- 参考风格关键词：韩漫写实、都市爽剧、系统流UI、喜剧Q版切换、阶层反差视觉化
```

## C. 场景氛围卡

### 【场景①：城市街道 / 光明银行外景 — 倾盆大雨】

```
- 主色调：深灰蓝 #2D3748 + 湿漉的黑灰
- 辅助色：路灯的昏黄 #F6AD55 / 银行招牌的冷白
- 光源：阴天漫射光（无直射阳光）+ 银行门口冷白日光灯 + 零星路灯暖黄
  色温整体偏冷（4000K以下）
- 空间感：开阔但压迫（低沉天际线、雨幕遮蔽远景、积水反射增加纵深）
- 材质倾向：湿漉混凝土路面（高反射）/ 金属栏杆（附着雨珠）/ 玻璃门面（雨痕流淌）
- 氛围关键词：压抑、孤独、冰冷、挣扎、困顿
- 叙事元素：
  · 银行招牌"云海省三台市光明银行XX分行"（建立地理信息）
  · 路面积水中模糊的银行招牌倒影（暗示现实与幻象的边界）
  · 被风吹翻的垃圾桶（无人在意的混乱）
- 伏笔元素：银行招牌上"光明"二字在雨中微微发光（后续总行场景的呼应）
- 情绪变化方案：
  · EP001前段：倾盆大雨，天色昏暗
  · EP001后段（押运车出来时）：雨过天晴，阳光初现（命运转折的视觉映射）
```

```

- 字体层级：
  · 系统提示文字：等宽无衬线（类Orbitron/Rajdhani），#00B0FF，带轻微发光效果
  · 数字（金额/数据）：加粗等宽，#FFD700，带脉冲发光动画
  · 系统"叮！"提示：大号粗体，缩放弹入 + 光波扩散
- 弹出动画：
  · 触发音效"叮！"→ 画面微Glitch（0.2秒）→ 像素化组装（从中心向四周扩散，0.5秒）
    → 面板稳定显示 → 文字逐行扫描打出（打字机效果）
  · 消失：面板碎片化分解 → 粒子飘散 → 消失
- 与角色空间关系：
  · 漂浮在许正阳面前约50cm处（只有他能看到，其他角色无反应）
  · 面板略微跟随许正阳视角转动
- 交互反馈：暂无交互场景，纯信息推送

【字幕/对白系统】
- 角色台词：标准圆角气泡，白底黑字，气泡尾部指向说话者

- 音效字（漫画式）：如"啪嗒"——粗体、动态变形、带运动线

【转场特效库】
- 系统触发转场：画面整体Glitch抖动（0.3秒）+ 短暂黑屏 + 新画面像素化组装进入
- 记忆闪回：画面从中心碎裂 → 碎片翻转显示回忆画面 → 去饱和 + 颗粒感 + 画幅缩窄（加黑边）
- 场景空间跳转：前一场景最后画面元素（如门框/车窗框）作为遮罩擦除进入新场景
- Q版切换：画面"噗"一声缩小到中心 → Q版变形角色从缩小的画面中弹出 → 背景变为简化纯色
- 时间跳跃：光影快速流动（太阳移动轨迹加速）+ 数字时钟翻页
- 情绪重击：画面突然静止0.5秒 → 轻微放大 → 恢复（用于"卧槽"反应时刻）

- 雨后天晴：水汽蒸腾粒子（地面上升）+ 阳光光柱中的微尘 + 偶现彩虹光晕
- 贵宾厅：水晶吊灯折射的七彩光点缓慢移动在墙壁和地面上
- 服装店：暖白射灯光柱中的极微灰尘 + 丝绒面料的微光闪烁
```
