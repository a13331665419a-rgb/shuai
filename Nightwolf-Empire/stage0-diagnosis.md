## 二、制作需求判断

### 对话 vs 动作占比

- **对话驱动为主**（约 75%），辅以身体对抗、物理亲密、喜剧动作
- 对话类型：公开审判式对话、双人私密博弈、言语交锋（Violet 的嘴炮是核心卖点）
- 动作类型：掐脖子、扇耳光、膝击、拉扯、躲藏、摔倒

### 特殊制作需求

1. **双人桌下空间**（EP04–05）：极窄空间内的身体喜剧 + 暧昧，需要特别的构图策略
2. **脱衣猜谜游戏**（EP09）：需要"战略性遮挡"手法（参考 *Why Women Kill*），既暴露又不违规
3. **红线 VFX**：需贯穿全剧作为"命定伴侣"视觉符号，风格需统一
4. **闪回处理**（EP03）：婚礼场景需要明显的时间跳跃视觉标记（色调变暖/柔焦/Super字幕）

### 画面比例

- **9:16 竖屏**（北美短剧标准）
- 竖屏对本剧尤其有利：人物面部特写、双人对峙的压迫站位、桌下窄空间的幽闭感

## 三、核心视觉卖点判断

### 观众来看什么？

1. **Violet 的嘴炮反击**：一个没有狼、被丈夫嫌弃的女主，用语言把所有人怼到哑口无言
2. **Nicholas 的危险吸引**：冷酷国王在她面前逐渐失控，克制中的欲望比直接表达更撩人
3. **命定伴侣的秘密纽带**：只有两人知道的秘密，在所有人面前维持"陌生人"表演
4. **公开打脸 Kylan**：渣男被反复羞辱，观众获得替代性爽感
5. **身份差的视觉冲击**：狼人帝国最高权力者 vs 唯一没有狼的贵族女儿

### 最该拍重的场面类型

- 公开审判 / 当众对峙
- 双人密室博弈（权力 + 情欲）
- Violet 的反击瞬间（语言 + 动作）
- Nicholas 失控的微表情
- 渣男被打脸的群体反应

## 四、生产参数建议

| 参数 | 建议值 |
|------|--------|
| 画面比例 | 9:16 竖屏 |
| 美术风格 | 写实真人 + 戏剧性强化 + 高级商业短剧感 |
| 光影基调 | 高对比戏剧光（议事厅）/ 暖金色（回忆）/ 冷蓝（夜间血约） |
| 质感 | 轻胶片颗粒 + 商业广告级皮肤质感 |
| AI 生成工具 | 可灵 3.0 / 即梦 Seedance 2.0（待确认） |
| 分镜精度 | 镜头级（逐 shot 设计） |
| 视觉风格参考 | *Why Women Kill* 的构图遮挡 + *Bridgerton* 的豪门质感 + ReelShort 狼人剧的 VFX 风格 |

## 五、待确认事项

1. **AI 生成工具确认**：可灵 3.0 还是即梦 Seedance 2.0？还是双平台适配？
2. **角色族裔/外貌方向**：剧本未明确角色种族，需确认是否偏白人欧美、混血、还是开放
3. **裸露尺度**：EP09 的脱衣猜谜游戏，平台允许的最大尺度是什么？影响遮挡策略
4. **狼人变身 VFX**：当前剧本只有"眼睛变红"和"红线"，后续是否需要完整变身？
5. **后续集数**：当前 12 集是否为全部？是否有后续 EP13+ 的剧本？
6. **Queen Mother 线**：EP12 开启了新冲突线，是否在后续集数中延续？需了解走向以设计卡点
7. **Lucy 的螃蟹阴谋**：EP12 暗示 Lucy 会利用 Violet 的过敏，具体执行方式需确认
