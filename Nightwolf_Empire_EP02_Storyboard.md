# 夜狼帝国 Nightwolf Empire — EP02 导演逐镜头分镜脚本

## 剧本解读与节拍拆解

**本集核心戏剧目的**：Kylan公开提离婚 → Violet当众反击 → Nicholas以王权否决 → 走廊暴力 + Violet强硬反击

**题材类型**：女频强情绪 + 公开羞辱 + 女主反击 + 权力审判

**核心驱动力**：公开羞辱 → 爽感反击 → 权力审判 → 身体压制 → 反击升级

**节拍拆解**：
1. Kylan陈述离婚理由——引入Lucy（众人震惊）
2. Nicholas不回应Kylan，直接问Violet（权力转移）
3. Violet的精彩反击——犀利、幽默、不卑不亢
4. Nicholas以Alpha Tone压制全场（绝对权力爆发）
5. Nicholas否决离婚（权力审判完成）
6. 走廊——Kylan掐Violet脖子（暴力升级）
7. Violet甩耳光（女主第一次物理反击，强卡点）

**预估镜头数**：70–90镜
