# 夜狼帝国 Nightwolf Empire — EP06 导演逐镜头分镜脚本

## 剧本解读与节拍拆解

**本集核心戏剧目的**：EP05卡点解除（Violet用面纱掩护脱身）→ Violet犀利反击Lucy → Selene维护Violet → Kylan与Lucy被赶出 → Violet独自面对Nicholas → Nicholas设定三天后规则

**题材类型**：女频强情绪 + 口舌战斗 + 权力反转 + 暧昧对峙

**核心驱动力**：爽感反击 → 关系误解 → 权力压制 → 危险吸引深化
