# 夜狼帝国 Nightwolf Empire — EP08 导演逐镜头分镜脚本

## 剧本解读与节拍拆解

**本集核心戏剧目的**：Sam打断暧昧→ Celina情人闯入危机→ Nicholas把Violet按在王座上伪装亲吻→ Celina看到后心碎离去→ Nicholas下令处死卫兵→ Violet被迫面对Nicholas的残忍→ 最后的"Turn me on"挑战——付费卡点

**题材类型**：女频强情绪 + 暧昧危机 + 伪装亲密 + 暴力暗流 + 情色悬念

**核心驱动力**：暧昧升级 → 危险伪装 → 权力暴力 → 情色悬念卡点
