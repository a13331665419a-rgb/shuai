# 夜狼帝国 Nightwolf Empire — EP04 导演逐镜头分镜脚本

## 剧本解读与节拍拆解

**本集核心戏剧目的**：月亮祭坛偶遇 → Violet偷听Nicholas的致命秘密 → Nicholas掐脖试图灭口 → [VFX]红线缠绕揭示Fated Mate → Selene揭示"喝她血续命"→ 关系质变

**题材类型**：女频强情绪 + 超自然命运 + 危险吸引 + 暧昧窒息感

**核心驱动力**：危险吸引 → 真相揭露 → 命运绑定 → 身体压制的暧昧

**节拍拆解**：
1. 月亮祭坛场景建立——神秘、仪式感
2. Violet跪地祈祷——脆弱真实的一面
3. 项链断裂滚入桌下——命运的牵引
4. Nicholas与Selene密谈进入——Violet被困桌下
5. 偷听关键秘密——Nicholas的弟弟、狼毒、解药
6. Nicholas发现Violet——掐脖灭口——极致危险吸引
7. [VFX]红线缠绕双方手腕——Fated Mate确认
8. Selene揭示"喝她的血"——新规则建立
9. Violet的"you can drink all you want"——荒诞勇敢的求生
