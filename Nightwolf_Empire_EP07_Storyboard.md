# 夜狼帝国 Nightwolf Empire — EP07 导演逐镜头分镜脚本

## 剧本解读与节拍拆解

**本集核心戏剧目的**：Garrick粗暴带走Violet → 议会大厅放血仪式 → Nicholas半裸诱惑 → Violet偷看被威胁 → 血液的甜蜜——Nicholas差点亲吻伤口

**题材类型**：女频强情绪 + 肉体吸引 + 权力支配 + 情色擦边 + 暧昧蓄压

**核心驱动力**：身体压制 → 情色张力 → 危险吸引深化 → 血缘/命运纽带

**节拍拆解**：
1. 夜——Anna昏迷、Garrick粗暴劫走Violet
2. 议会大厅空旷夜场——刀落地，放血命令
3. Nicholas半裸亮相——Violet偷看
4. 割掌放血——[VFX]红线再现
5. Nicholas嗅到血的甜味——欲望失控边缘
6. Nicholas差点亲吻她的手——暧昧高潮→被打断
